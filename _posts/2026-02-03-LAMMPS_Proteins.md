# Using open-source tools to simulate a protein
## Tools utilised
 - Protein Data Bank (RCSB)
 - Visualization tools like ChimeraX, Avogadro and VMVD
 - pdbfixer
 - AmberTools which provides Antechamber and (x)(t)leap
 - AMBER TO LAMMPS (AMBER2LAMMPS) python code
 - LAMMPS (sample input file included)
 - OVITO for visualization
